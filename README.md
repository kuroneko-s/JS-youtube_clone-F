# Wetube_2021

구버전 영상이 사라져서 강제로 진행중

global router
/ -> Home
/join -> Join
/login -> Login
/search -> Search

user router
/users/:id -> See User
/users/logout -> Log Out
/users/edit -> Edit my Profile ( Login )
/users/remove -> Delete my Profile ( Login )

video router
/videos/:id -> See Video
/videos/:id/edit -> Edit my Video
/videos/:id/delete -> Delete my Video
/videos/upload -> Upload Video
